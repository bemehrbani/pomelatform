# Pometools / Pomegroup Website

**Status**: Active
**Type**: Next.js 16 Web Application
**Location**: `pomegroup-website/`

## Description
This is the main public website or dashboard portal for Pomegroup. It was previously located at the root of the repository.

## Tech Stack
-   **Framework**: Next.js 16 (App Router)
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS (via PostCSS/Next.js default)
-   **Package Manager**: npm/yarn/pnpm/bun

## Key Commands
-   `npm run dev`: Start development server
-   `npm run build`: Build for production
